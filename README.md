# RuaKey — Releases

This repo hosts **RuaKey** release artifacts only (Sparkle `appcast.xml` + signed `.dmg`).
The app's auto-updater reads `releases/latest/download/appcast.xml`.

- App: a local-first macOS keyboard-analytics menubar app. Your input data never leaves your Mac.
- Install: download the latest DMG, drag to Applications. First launch: right-click → Open (ad-hoc signed).
